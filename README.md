# EARtist
Evolutionary artist and their hundreds of shapes.

## Quick Start
```bash
git clone https://github.com/PkuCuipy/eartist.git
cd eartist
cargo run --release
```

Result images will be saved in `./src/result/generation_best_{datetime}/` directory.
